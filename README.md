# potflow

Interatomic potential workflows (PotFlow).


## Installation 

- `potflow` requires Python 3.9+
 ```
 git clone https://github.com/mjwen/potflow.git
 cd potflow 
 conda create --name potflow 
 conda activate potflow 
 conda env update -f environment.yml
 pip install -e . 
 ```
